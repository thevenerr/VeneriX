VeneriX

A lightweight game engine built from scratch in modern C++. OpenGL rendering, a simple scene/entity/component model, and the basics you need to start building — without the bloat.

C++17 OpenGL 3.3 v0.1-alpha
↓ Download SDK Read the docs

What's inside

Scene · Entity · Component

A small, explicit object model. No hidden ECS magic — just classes you can read and extend.

Automatic updates

The engine drives the update loop for you. Entities and components update themselves every frame.

Time scale control

Built-in TimeStep system with adjustable time scale — pause, slow motion, or fast-forward with one call.

Instanced rendering

Draw hundreds of cubes in a single draw call — ideal for tiled floors and voxel-style worlds.

Audio built in

Music and one-shot sound effects out of the box, powered by miniaudio.

ImGui debug tools

Dear ImGui is wired in from the start, so your debug UI is one ImGui::Begin() away.

Built on

OpenGL GLFW GLAD GLM Dear ImGui miniaudio
